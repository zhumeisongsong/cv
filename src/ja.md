## 📌 サマリー

**シニアフロントエンドエンジニア**

10 年以上の経験を持ち、スケーラブルでユーザーフレンドリーなウェブアプリケーションの設計と構築を担当。

**Front-End:**

- React の経験 9 年以上、TypeScript の経験 7 年以上
- スケーラブルで保守性の高いフロントエンドアーキテクチャの設計に精通（モノレポ、マイクロフロントエンド、クリーンアーキテクチャなど）
- クリーンなコードを書きながら、ユーザーフレンドリーなインターフェースを開発
- ドメイン駆動設計（DDD）とデータモデリングの深い理解
- パフォーマンス最適化の専門知識（ESM ツリーシェイキング、コード分割、レイジーローディング、画像最適化、LCP の改善など）
- フロントエンドエンジニアリングのベストプラクティスの知識（Webpack、Vite、Eslint、型チェック、テスト、CI/CD など）

**Back-End:**

- NestJSの実務経験3年以上
- API（GraphQL、RESTful API）の構築に関する実務経験
- フロントエンドアプリケーションとのシームレスな統合に精通
- クリーンアーキテクチャおよびクリーンコードの原則に関する専門知識
- イベント駆動アーキテクチャの経験

**Product:**

- 投資家やステークホルダーへのアイデア提案の経験

- チームプレーヤーとして、チームメンバー（プロダクトオーナー、デザイナー、バックエンドエンジニアなど）と協力して目標を達成

::: details ソフトウェアエンジニア @[Atrae](https://atrae.co.jp/) _(2021.12 ~ · 3 年+)_

Atrae は People Tech 分野の日本の IT 企業で、私は Wevox プロダクトチームの一員です。[Wevox](https://get.wevox.io/)は、組織の能力を向上させることを目的とした B2B SaaS ウェブサービスです。Wevox は 3,390 以上の組織に採用され、これまでに 2.83 億件以上の調査回答データを集めています。

React/Next.js/TailwindCSS/Jotai/Valtio/Immer/Nx Monorepo/TypeScript/NestJS/NodeJS/GraphQL/Restful API/Axios/Urql/Docker/AWS/GitHub Actions

- ドメイン層とアプリケーション層をプレゼンターから分離することで、プレゼンターの肥大化問題を解決。これにより、変更範囲の管理が容易になり、テスト時に必要なモックメソッドの数が大幅に減少

- NestJS を使用して、GraphQL ベースの Query と Mutation を開発し、クエリのパフォーマンスを最適化
- Persisted Queries を使用してリクエストサイズを削減し、帯域幅の消費が減少。サーバーが毎回クエリ内容を解析・検証する際のパフォーマンスオーバーヘッドを解消
- 10MB を超える大きな JS ファイルに Gzip 圧縮を適用し、初期ページのロード時間を大幅に短縮
- デザイナーと協力し、Material UI に基づいた再利用可能な UI コンポーネントライブラリ（LoadingButton、Steps、Rate、Spin、Selectable Table などのコンポーネントを含む）を開発し、チームの開発効率とコードの一貫性を向上

## 🎨 目標やビジョン

私は、シンプルさには力があると信じており、物事をシンプルにする助けとなることを実践するのが好きです。

新しい問題を作り出して古い問題を解決することは、一見進歩のように思えるかもしれませんが、本質的にはより大きな円を描いて同じ場所を回っているに過ぎないことが多いです。

私は、ページや API の構築といった具体的なタスクを完了するだけでなく、**ローカップリング・ハイコヘージョン**のソフトウェアアーキテクチャの探求にも力を入れたいと考えています。

私が求める役割には以下の内容が含まれていることを希望します：

- 技術的負債の解消
- 複数のサービスにわたる複雑性の管理
- システムパフォーマンスデータの可視化と改善

これらの分野は、複雑さに取り組み、影響力のある解決策を提供するという私の強みに合致しています。

私の仕事への**アプローチ**は以下の通りです：

- 意義のある課題を選ぶ（価値創造を目指すため）
- シンプルな問題を徹底的に分析する（リスクを管理し、将来に備えるため）
- 複雑な問題を行動でシンプルにする（品質の高い製品を提供しつつ、コストを抑えるため）
- 複雑さを簡潔に説明する（知識の確実な移転を確保するため）
